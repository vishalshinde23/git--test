# Calculator Project

This project is a simple calculator built using HTML and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Project Structure

```
calculator-project
├── src
│   ├── index.html      # HTML structure for the calculator
│   └── script.js       # JavaScript functionality for the calculator
└── README.md           # Documentation for the project
```

## How to Use

1. Open the `src/index.html` file in a web browser.
2. Enter numbers in the input fields.
3. Click on the operation buttons (e.g., +, -, *, /) to perform calculations.
4. The result will be displayed in the designated area.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- User-friendly interface for easy interaction.
